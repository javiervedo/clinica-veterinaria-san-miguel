# Diseño técnico y propuesta tecnológica

## Propuesta de solución

Se propone una aplicación web con arquitectura cliente-servidor y base de datos centralizada.

## Arquitectura

- frontend web
- backend API REST
- base de datos PostgreSQL

## Tecnologías seleccionadas

### Backend
- Node.js
- Express
- PostgreSQL
- pg
- Jest
- Supertest

### Frontend
- React
- Vite

## Justificación

- bajo coste de adopción
- facilidad de despliegue
- buena mantenibilidad
- amplia comunidad y documentación
- escalabilidad suficiente para una clínica de tamaño medio

## Principios técnicos

- separación por capas
- validación de datos en backend
- control de errores centralizado
- estructura modular
- cobertura de pruebas básica
