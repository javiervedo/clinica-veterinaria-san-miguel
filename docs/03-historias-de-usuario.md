# Historias de usuario

## HU01. Alta de propietario
Como personal administrativo, quiero registrar un propietario para asociarle mascotas.

**Criterios de aceptación**
- debe poder registrarse nombre y teléfono
- el propietario queda disponible en el sistema

## HU02. Alta de mascota
Como personal administrativo, quiero registrar una mascota asociada a un propietario para gestionar su información.

**Criterios de aceptación**
- no puede crearse sin propietario
- debe registrarse al menos nombre y especie

## HU03. Crear cita
Como administrativo, quiero crear una cita para organizar la agenda de la clínica.

**Criterios de aceptación**
- no debe permitirse solape para el mismo veterinario
- la cita queda visible en la agenda

## HU04. Consultar historial
Como veterinario, quiero consultar el historial clínico de una mascota para atenderla correctamente.

**Criterios de aceptación**
- se muestran consultas, tratamientos y observaciones previas

## HU05. Registrar tratamiento
Como veterinario, quiero registrar un tratamiento para hacer seguimiento clínico.

**Criterios de aceptación**
- debe guardar pauta, fecha de inicio y estado

## HU06. Consultar recordatorios
Como auxiliar, quiero ver seguimientos pendientes para contactar a los clientes.

**Criterios de aceptación**
- el sistema muestra recordatorios por fecha y estado
