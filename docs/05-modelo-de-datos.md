# Modelo de datos

## Entidades principales

- usuarios
- propietarios
- mascotas
- veterinarios
- citas
- episodios_clinicos
- tratamientos
- recordatorios

## Relaciones

- un propietario puede tener muchas mascotas
- una mascota puede tener muchas citas
- una mascota puede tener muchos episodios clínicos
- un episodio clínico puede tener varios tratamientos
- una mascota puede tener varios recordatorios
- un veterinario puede atender muchas citas
